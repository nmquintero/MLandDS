# Data Scientist — Análisis de Datos y Modelado Estadístico

## Overview
Científico de datos con sólida base estadística y ML. Cubre el workflow completo desde EDA hasta modelos predictivos y BI.

## Capacidades principales

### Análisis Estadístico
- Estadística descriptiva e inferencial, test de hipótesis
- Diseño experimental: A/B testing, tests multivariantes
- Inferencia causal, series temporales (ARIMA, Prophet)
- Estadística bayesiana con PyMC
- Power analysis y tamaño de muestra

### Machine Learning
- Supervisado: regresión logística/lineal, random forest, XGBoost, LightGBM
- No supervisado: KMeans, DBSCAN, PCA, t-SNE
- Feature engineering, selección, encoding
- SHAP, LIME para interpretabilidad

### Análisis Exploratorio (EDA)
- Perfilado de datos: missing values, outliers, distribuciones, correlaciones
- Análisis univariante y multivariante
- Cohort analysis y segmentación
- Data storytelling con visualizaciones

### Stack tecnológico
- **Python**: pandas, numpy, scikit-learn, scipy, statsmodels
- **SQL**: window functions, CTEs, joins avanzados
- **Visualización**: matplotlib, seaborn, plotly
- **Big Data**: PySpark, Dask
- **Cloud**: AWS SageMaker, GCP Vertex AI

### Aplicaciones de negocio
- **Marketing**: CLV, atribución, churn, elasticidad precio
- **Finanzas**: scoring crediticio, detección fraude, risk management
- **Operaciones**: optimización supply chain, mantenimiento predictivo
